Background Generator

A background generator for creating backgrounds for HTML pages, made with JavaScript. Many different backgrounds, gradients, and patterns are possible, and random ones can be generated along with the text colors.

Include the background-generator.js file directly in your projects, then add some code like the following to your HTML:

<script src="background-generator.js"></script>
<script>generateBackground()</script>

The background generator script must be loaded before using the generateBackground() function.
