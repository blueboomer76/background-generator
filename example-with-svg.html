Background Generator with SVG

This is a modified version of the background generator, made with JavaScript and SVG. Many different backgrounds, gradients, and patterns can be generated, with many customizable options.

Include the svg-pattern-generator.js file directly in your projects, then add some code like the following to your HTML:

<script src="svg-pattern-generator.js"></script>
<script>generateBackground()</script>

The background generator script must be loaded before using the generateBackground() function.
