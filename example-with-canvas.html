Pattern Generator with Canvas

This is a modified version of the background generator, made with JavaScript and the HTML5 Canvas API. Many different backgrounds, gradients, and patterns can be generated, with many customizable options.

Include the canvas-pattern-generator.js file directly in your projects, then add some code like the following to your HTML:

<canvas id="myCanvas"></canvas>

<script src="canvas-pattern-generator.js"></script>
<script>
	const myCanvas = document.getElementById("myCanvas");
	const ctx = myCanvas.getContext("2d");
	const gradient = generateCanvasPattern(ctx, myCanvas.width, myCanvas.height);

	ctx.fillStyle = gradient;
	ctx.fillRect(0, 0, myCanvas.width, myCanvas.height);
</script>

The background generator script must be loaded before using the generateCanvasPattern() function. An error will be thrown if the first argument passed to the function is not a CanvasRenderingContext2D.
